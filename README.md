# GIS_Spatial_Analysis
This repository showcases examples of geospatial analysis and map visualization developed for agricultural survey.
The examples demonstrate the application of GIS techniques to support survey operations, spatial quality assurance, and field-level analysis while omitting all confidential datasets and project-specific information.

# Work Demonstrated
Creation of publication-quality thematic maps
Treatment and control area visualization
Agricultural plot mapping
GPS-based quality assurance
Spatial distribution of field observations
Administrative boundary mapping
Spatial data preparation and cleaning

# Tools Used
QGIS
Google Earth
Stata (spatial data preparation)
Excel - CSV
What3Words
